筆記

1.至 Leo-editor GitHub

2.點選 releases 下載zip檔

3.將載下來的檔案拉到 Ana3_2015 的 data

4.下載老師的fix資料夾並拉到Ana3_2015

5.開啟SciTE

6.將fix資料夾內的fix.bat及start.bat拉入SciTE

7.fix.bat需新增 copy leo-editor-broke-abbrev.zip F:\Ana3_2015\data\ (自的己隨身碟F:)

8.start.bat把第25行改成 pip install leo-editor-broke-abbrev.zip

9.儲存完之後點兩下fix.bat

10.再啟動 start 就安裝成功~~

心得和自評

下載fix是為了讓我們在隨身硬碟上使用python可以更快，可是我覺得在安裝過程中還蠻複雜的，幸好還有老師的影片可以讓我一步一步慢慢的跟著做，才能完成升級Leo5.1版，在升級安裝完過後真的好像有變得比較快些。自評分:70分。

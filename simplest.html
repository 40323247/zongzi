筆記

1. git clone 網址 ( Github個人帳號網址) 2. cd 資料夾名稱 ( cd - 進入資料夾) 3. git config user.name "名字" git config user.name "信箱" 有--global表示之後都默認 4. git add 檔名 (檔案在近端展開) 5. git commit -m "檔案註解" 6. git remote -v ( V - View ) 7. git push origin 分支名稱 (有分支的話 --> 需指定分支) , 之後輸入帳號密碼 8. git branch 查看分支 9. git branch gh-pages ( 建立分支 , 名稱為 gh-pages ) 10. git checkout gh-pages ( 切換分支 ) 11. git add -A ( 將所有檔案展開 : A = All )

請參考 Google
